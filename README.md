### 브랜치 네이밍
⚙️ 네이밍 패턴

제작자 이름/간단한 설명

Ex) '로그인 기능' 이슈를 구현하는 브랜치를 생성하는 경우, 브랜치 이름을

*yoona-j/loginPage* 로 작성한다.

### 커밋 메시지

commit message는 무조건 자신이 구현한 부분에 대해 정확하게 서술!

Ex) '회원가입' 기능을 구현한 경우, 커밋 메세지를

*make join page and create member DB* 로 작성한다.
